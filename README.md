# climate_history
Analysis of historical global climate data including temperature, sea ice, and sea level.
NASA GISS, NSIDC, NOAA 데이터에 기록되어 있는 지구의 평균 온도, 해수면의 깊이, 북극과 남극의 해빙 변화량을 사용하여 과거에서부터 현재까지 지구가 어떻게 변했는지 살펴봤습니다.
